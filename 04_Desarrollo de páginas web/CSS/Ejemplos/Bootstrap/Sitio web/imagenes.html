Imágenes con Bootstrap

Documentación de imágenes en Bootstrap

Imágenes responsive

Bootstrap 5 no adapta el tamaño de las imágenes automáticamente. Para hacer una imagen adaptable se debe añadir la clase .img-fluid a cada imagen que se desee que se comporte de manera responsive.

Otros efectos

Además, se puede usar .img-thumbnail para dar a una imagen una apariencia de foto polaroid con borde redondeado de 1px.
