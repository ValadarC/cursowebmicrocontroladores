Estilos CSS siguiendo el Box Model

Un estilo en línea es un método para insertar el lenguaje de estilo de página directamente dentro de una etiqueta HTML. Esta manera de proceder no es totalmente adecuada.

El incrustar la descripción del formato dentro del documento de la página web, a nivel de código, se convierte en una manera larga, tediosa y poco elegante de resolver el problema de la programación de la página.

Funciona pero no es todo lo claro que debería ser.

La solución más recomendable para maquetar correos electrónicos, es utilizar CSS dentro de los propios elementos (online)
